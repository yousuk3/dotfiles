# dotfiles

## Overview

This repository contains scripts to set macOS Sonoma.

## Install

Download installation materials.

```shell
% cd ~ && git clone https://github.com/yousuk3/dotfiles
```

Set macOS.

```shell
% cd ~/dotfiles && make
```
